# Contributing

Linux Made Sane accepts contributions that improve the Community Edition, documentation, tests, packaging, and safe public release assets.

Do not submit secrets, customer data, private repository details, proprietary Pro or Enterprise code, private infrastructure URLs, or anything you do not have the right to share.
